# DNN_HighDimPDEs  this depository contains all codes and input data 
for the numerical results reported in the paper "Deep Neural networks for solving 
high-dimensional parabolic partial differential equations" (HJB - Hamilton-Jacobi-Bellman, BS-Black-Scholes) 
by four methods: PINN, DeepBSDE, DeepMartNet, Shotgun.
